# Indian Facial Expression Dataset

### **[Github](https://ifexd.github.io)**
### **[PyPi - project](https://pypi.org/project/ifexd/)**

**Indian Facial Expression Dataset**
* 12K images including three views - Front, Left, Right
* Images from 345 Indians
* 7 Expressions + Natural pose:
  * A: Natural
  * B: Happy
  * C: Surprise
  * D: Sad
  * E: Anger
  * F: Contempt
  * G: Disgust
  * H: Fear
